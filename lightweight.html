About Me

I am a Computer Science major with an interest in game development, interaction design, and the building of intuitive digital experiences. I enjoy projects that combine programming with creativity, whether it's developing game levels, designing user interfaces, or building tools like my custom Discord bot.

I'm always trying new technologies and finding ways to make experiences feel smoother, more expressive, and more fun for people to use. I tend to approach problems quietly and thoughtfully, learning through hands-on experimentation and building things that reflect my interests.

AwesomeBot99999 - Multipurpose Discord Bot (8/3/2021-Present)

  • Developed a large-scale Discord bot using Node.js and Discord.js with over 110 modular commands.
  • Implemented a persistent database system and optimized backend logic for reliability and scalability.
  • Containerized the application with Docker for consistent 24/7 uptime.
  • Open-sourced the project with documentation and versioning.
Takeaways:
  • Improved backend development, database management, and real-time event handling skills.
  • Gained experience maintaining a long-term software project used by real users.

Personal Portfolio Website (In Progress)

  • Built using plain HTML, CSS, and JavaScript without frameworks to ensure broad compatibility.
  • Designed with responsive layout and accessibility features such as dark mode, high contrast, and dyslexia-friendly options.
  • Deployed through GitHub Pages with version control for iterative updates.
  • Tested on legacy browsers and systems (including Chrome on Windows XP, Opera 9 on Nintendo Wii, and Mosaic 1.0 on Windows 95) to explore compatibility constraints.
  • Takeaways:
    • Developed practical experience building accessible, lightweight web pages that function without modern frameworks.
    • Gained insight into how outdated browsers interpret or fail to interpret modern HTML, CSS, and JavaScript standards.

Huffman Coding Compressor (3/29-4/10/2024)

  • Created a text compression and decompression tool in C++ using the Huffman coding algorithm.
  • Implemented binary tree construction and bit-level file encoding/decoding.
  • Achieved compressed file sizes of about 60% of the originals for large text files.
Takeaways:
  • Gained experience with data compression, tree structures, and bitwise manipulation.
  • Learned modular C++ design through organized header and driver files.

Weekly Carbon Footprint Calculator (11/27-12/8/2024)

  • Collaborated in a three-person team to build an interactive weekly carbon footprint calculator using MIPS Assembly.
  • Classmates focused on implementing the input and calculation routines, while I designed and programmed the animated bar-graph visualization that displays the results.
  • Synchronized the bar graph with the computed emissions so users can quickly see how different activities compare week to week.
Takeaways:
  • Gained experience turning low-level numeric output into a readable visual representation using assembly language.
  • Learned how to coordinate roles in a small team project while focusing on a specific feature area.
