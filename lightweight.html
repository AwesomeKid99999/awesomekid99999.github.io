Featured Projects

STEM Tutor - Offline Learning Platform (9/13-9/14/2025)

  • Designed the UI/UX for an offline STEM tutoring tool during a 24-hour hackathon.
  • Built an accessible interface with Tailwind CSS and worked with a small team of four to wire up an AI library for natural-language feedback.
  • Demo'd the whole thing to judges and helped the project snag a design award.
What I learned:
  • Got good at rapidly throwing together UIs when the clock is ticking.
  • Figured out how to design for offline and low-connectivity environments.

AwesomeBot99999 - Multipurpose Discord Bot (8/3/2021-Present)

  • Built a Discord bot using Node.js and Discord.js with over 110 commands that contains loads of features.
  • Set up a persistent database system and optimized the backend to keep things reliable and scalable.
  • Containerized it with Docker so it could run 24/7 on its own.
  • Open-sourced the whole thing with docs and proper versioning.
What I learned:
  • Leveled up my backend development, database management, and real-time event handling skills.
  • Got real experience maintaining a project that actual people use long-term.

Personal Website (In Progress)

  • Built with plain HTML, CSS, and JavaScript without relying on frameworks. Keeps things lightweight and works everywhere.

Huffman Coding Compressor (3/29-4/10/2024)

  • Built a compression tool in C++ using the Huffman coding algorithm.
  • Implemented binary tree construction and bit-level file encoding/decoding to compress files.
  • Got files down to about 60% of their original size for large text files.
What I learned:
  • Got hands-on with data compression, tree structures, and bitwise manipulation.
  • Learned how to write modular C++ code with organized headers and driver files.

Weekly Carbon Footprint Calculator (11/27-12/8/2024)

  • Worked with two classmates to build a carbon footprint calculator in MIPS Assembly language.
  • While they handled input and calculations, I built the animated bar-graph visualization to show the results.
  • Synced the bar graph with the emissions data so users could easily compare activities week to week.
What I learned:
  • Figured out how to turn numeric output into readable visuals using low-level assembly language.
  • Got experience dividing up tasks in a small team and owning a specific feature area.
